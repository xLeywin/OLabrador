# OLabrador

> Website desenvolvido para a disciplina **Linguagem de Programação II** da **FATEC Americana-SP**

---

## 📋 Sobre o Projeto

**OLabrador** é uma aplicação web desenvolvida como projeto acadêmico na FATEC Americana-SP. O site foi construído utilizando **ASP.NET** com **C#** no back-end e **CSS** para estilização da interface.

🌐 **Deploy:** [wendell.somee.com](http://wendell.somee.com/)

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Uso | Proporção |
|---|---|---|
| C# | Lógica de back-end | ~48% |
| ASP.NET | Framework web | ~26% |
| CSS | Estilização | ~26% |
